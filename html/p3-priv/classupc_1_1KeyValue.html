Detailed Description

Class to tranform parameters in a string format as a key/value table

string s = "A=3; B=hola; lista=3,2,5;" KeyValue kv(s); cout << kv("A") << '
'; cout << kv("B") << '
'; cout << kv("lista") << '
';

This will show "3" "hola" "3,2,5"

Furthermore, it can transform the result to int, float, vector.

E.g.: int i; kv.to_int("A",i); vector<float> x; kv.to_vector("lista", x);
