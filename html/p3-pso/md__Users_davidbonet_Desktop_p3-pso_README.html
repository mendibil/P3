Técnica de preprocesado: center clipping

Hemos optado por el center clipping para realizar el preprocesado de la señal ya que es una técnica simple pero a la vez muy efectiva. Es un proceso no lineal que conserva la estructura periódica de la señal a la vez que intensifica la estructura armónica y reduce el ruido.

Para optimizarlo hemos establecido un offset para las muestras positivas y otro para las negativas. El valor de este offset es el resutado de multiplicar la potencia de todo el frame por un porcentaje. Este porcentaje es decisivo para obtener un buen resultado y por eso los hemos utilizado junto a la librería docopt_cpp para iterar diferentes posibilidades así pudiendo obtener los valores óptimos.

Técnica de postprocesado: filtro de mediana

Para el postprocesado hemos usado un filtro de mediana para así evitar el mayor numero de errores groseros como pueden ser la detección de un múltiplo o de un submúltiplo de la frecuencia real. Hemos analizado los resultados para diferentes longitudes del filtro y el que nos da un mejor resultado es el filtro de mediana de tres muestras.
